Some rules about washers that i made up while making them up:

  1. must be a single html file w/ no local dependencies/assets/etc.
  2. it can and should depend on internet websites for some content, primarily jquery, google fonts, jqueryui, and most importantly full window iframed youtube videos
  3. this is about fun, so best practices go out the window
  4. on the other hand, this is about fun, so why not test out some funny css experiment?
  5. should not take more than an hour to create. if u find yrself spending more than an hour on any project that means it has become a startup and that u should abandon it. rather tahn iterating over and over on a single idea, make something short simple and sweet. If you want to take that idea further, copy pasta and make a new washer. "sketching" as the Processing folks refer to it.
  6. must be a pleasant diversion or at least a room for critical thought. making a mean washer is just mean.
